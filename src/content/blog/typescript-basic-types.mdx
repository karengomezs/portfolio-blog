# Assign types to obejcts, arrays and variables

---

**_If you want to learn to assign different types to different data, here there are some ways_**

You can use [Interfaces](https://www.typescriptlang.org/docs/handbook/typescript-tooling-in-5-minutes.html#interfaces) and [Types](https://www.typescriptlang.org/docs/handbook/typescript-in-5-minutes.html), interfaces are like a "contract", it specifies the content, properties or functions to implement or meet the contract. Remember that **it does not contain** values or code in your functions. Let's do it!

### Snippet Variable

```ts
//First Example
let num1: number = 1;
const word: string = "hello";
let boo: boolean = true;
const arr1: Array<number> = [1, 2, 3];
let arr2: number[] = [4, 5, 6];
let obj2: any = { y: "hello" };
//const obj1: object = { x: 1 }; //not common way

//Second Example
type Word = string;
const word3: Word = "hello";
```

### Snippet Object

```ts
//1. Defining the Type

interface User {
  id: number;
  firstName: string;
  lastName: string;
}

//2. Here, you define the object and specify the type with --> :"User", you must use the same
//keys on the type and object declaration to meet the contract

const defaultUser: User = {
  id: 1,
  firstName: "karen",
  lastName: "gomez",
};

//3. And then.. go to implements in a function

const getUserFirstName = (user: User) => {
  return user.firstName;
};

console.log(getUserFirstName(defaultUser)); //"karen"

//Another Option with inline types

const getUserId = (user: {
  id: number;
  firstName: string;
  lastName: string;
}) => {
  return user.id;
};

console.log(getUserId(defaultUser)); //1
```

### Snippet Variable

```ts
//there are some ways to declare an array type and use it

let list2: Array<number> = [4, 5, 6];

// meeting the contract

```
